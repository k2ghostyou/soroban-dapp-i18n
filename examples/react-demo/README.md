# React Demo

This example demonstrates how to use `soroban-dapp-i18n` with a React app.

The demo includes:

- A language switcher for supported locales
- RTL-aware root rendering based on locale direction
- Locale-aware amount and date formatting

## Run

From the repository root:

```bash
npm install
# Use your preferred React setup to load examples/react-demo/src/App.tsx
```

In the demo, the app imports the local package sources with a relative path to `src/`.
